Magic Circle Loading Screen




Unity WebGL Player | Magic
Magic

























This was another one of my submissions for the Dev Club. The challenge was Magic, and so I decided I wanted to make an interactive magic circle that responded to key presses. When a key was pressed, the magic circle would change to that keys corresponding element. I think what I ended up creating was a very nice interactive loading screen for a magic style game. Each of the letters at the top refer to a key that can be pressed which will change the element of the circle. Can you figure out what they all are?
